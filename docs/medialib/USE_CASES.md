# MEDIA LIB

Application to manage media files (films and tv shows).

## USE CASES

### 1. Search a new 'Film' or 'TV Show' to download

1. User will type the film or tv show name to search.
2. System will search using an API like TVDB.
3. The results are shown.

### 2. User selects the search result to add to watchlist

1. User will select what's it's the search result that he wants to add to watchlist.
2. System will download all info for the result (if it's a tv show, it will download all seasons and episodes info).
3. System will add the result to watchlist.

### 3. User wants to see what's in watchlist

1. System will show all the items in watchlist.
